## 2차 전면 재조정 (AI 생성 패턴 제거 · 실제 결과물 중심)
1차 이후 "추상 AI영상·모노그램·Coming soon·약어 타일·반복 모션·개념 그래픽" 때문에 여전히 AI 느낌이 남는다는 피드백을 반영해 아래를 수정.

- **폰트**: 전 페이지 **Pretendard 단일**(Google Fonts·영문폰트 제거). 기계적 굵기 통일 폐기 → 편집 위계(Hero 300/강조 650·Statement 350·섹션 450·프로젝트명 650·서비스명 300/활성 600·본문 400).
- **Hero**: `<video>`·풀블리드 영상·MP4 전면 제거. **좌 텍스트 / 우 정적 데모 UI**(가짜 사명·수치 없이 콘텐츠 유형만 보여주는 실제형 화면) 3슬라이드 + 미세 scale-settle(1.03→1). 01 기업 데모UI / 02 브랜드 결과물 콜라주 / 03 업종 비교.
- **Selected Projects**: Coming soon·CO/HP/SH 모노그램·단색카드 제거 → **한 프로젝트=한 장면**(sticky + 스크롤 clip 전환). 유형·업종·목표·범위태그·대형 데모화면.
- **Production Services**: 모노그램 제거 → **7업종 각각 다른 화면 구조** 데모(데스크톱 hover 교체 / 모바일 인라인).
- **Brand Integration**: BR/CO/PH 약어타일 제거 → **결과물 콜라주**(대표 홈페이지+로고·촬영·카탈로그·모바일·앱 겹침).
- **Why HAOWEB**: 동일 형식 3반복·외곽선 글자 제거 → **세 장면 서로 다르게**(업종 좌우비교 / Before→Edit→Web / 아카이브 축적).
- **Search & AI**: 미래형 그래픽 없이 **SEO(검색결과)·AEO(Q&A)·GEO(관계 트리)** 실제 콘텐츠. "별도 광고상품 아님" 명시.
- **모션 축소**: 단어분할을 핵심 타이틀(메인 12개)만, 세부페이지 본문 즉시 표시. 반복 Fade-up 제거.
- **영문 라벨 축소**: 섹션 eyebrow 한글화. 영문은 히어로/프로젝트 카테고리·범위 태그만.
- **QA**: index 1440/1280/768/390/360 + 세부 8페이지 가로 오버플로 0, 콘솔 0, reduced-motion 지원.

## 무엇을 바꿨나
기존 하오웹이 "AI가 만든 느낌"(동일 카드 반복·번호 남용·영문라벨·반투명 UI·반복 Fade-up)이라는 피드백을 반영해, **사람이 편집·디렉션한 제작사 홈페이지**로 전면 재설계했습니다. 결과물·타이포·이미지·여백이 먼저 보이도록.

## 디자인 언어
- **컬러**: Black `#0D0C0B` · Ink `#16130E` · Off-white `#F2EFE8` · White · **HAOWEB Red `#E83817`(포인트만)**
- **타이포**: Pretendard. "크게+가늘게"(초대형 thin) ↔ 굵은 대형의 강한 대비. 영문 라벨/번호는 mono로 최소.
- **레이아웃**: 박스 카드 대신 **구분선 + 여백**. 화면마다 달라지는 구성(비대칭·좌우교차·콜라주·sticky).
- **모션**: 절제. 헤드라인 **단어별 split-text 마스크 리빌**, 스크롤 블록 리빌, 히어로 롤링/영상. 발광·파티클·글리치 없음. `prefers-reduced-motion` 지원.
- **명암 리듬**: 다크 히어로 ↔ 오프화이트 콘텐츠 교차.
