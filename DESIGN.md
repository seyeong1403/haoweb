# 하오웹 — 디자인 틀 (Design Framework)

브랜드의 시각 기준 문서입니다. 색·타이포·간격·컴포넌트의 **단일 소스**는 `css/tokens.css`이며,
`css/style.css`는 이 토큰만 참조합니다(하드코딩 금지).

---

## 1. 브랜드 앵커

| 항목 | 값 | 출처 |
|------|-----|------|
| 로고 | `assets/logo.png` — "hao" 클라우드 워드마크 (투명 배경) | 기존 사이트 haoweb.co.kr |
| 브랜드 코어 컬러 | **`#E83817`** (버밀리언/주홍) | 로고에서 추출 |

로고는 **글자가 투명 컷아웃**이라 밝은 배경에서 가장 잘 읽힙니다 → 헤더(페이퍼 배경)에 배치.
클라우드의 둥근 형태 = 브랜드 성격이 *따뜻하고 친근함* → 전체 라운드(둥근 모서리)로 반영.

---

## 2. 컬러 시스템 (3-레이어 토큰)

```
Primitive (원시값)  →  Semantic (용도)  →  Component (컴포넌트)
--red-500 #E83817   →  --color-primary  →  --btn-primary-bg
```

**팔레트 방향**: 워밍(버밀리언) + 쿨한 페이퍼 화이트의 온도 대비.
AI 디자인의 흔한 기본형(크림+세리프+테라코타 / 새까만 배경+단일 액센트)을 의도적으로 회피.

| 역할 | 토큰 | 값 |
|------|------|-----|
| 브랜드/CTA | `--color-primary` | `#E83817` |
| 호버 | `--color-primary-hover` | `#CB2C0F` |
| 하이라이트 배경 | `--highlight-bg` | `#FFF3F0` |
| 페이지 배경 | `--bg` | `#FAFAF9` (쿨 페이퍼) |
| 카드/표면 | `--surface` | `#FFFFFF` |
| 본문 텍스트 | `--text` | `#1B1815` (웜 잉크) |
| 보조 텍스트 | `--text-soft` | `#494540` |
| 다크 섹션 | `--ink-900` | `#1C1712` (진단 CTA·푸터에만) |

버밀리언은 CTA·하이라이트·강조에만 절제해서 사용(“AI가 선택한 순간”을 표시). 넓은 면은 중립색.

---

## 3. 타이포그래피

| 역할 | 폰트 | 용도 |
|------|------|------|
| 디스플레이·본문 | **Pretendard** | 제목/본문. 굵기·크기 대비로 위계 |
| 유틸리티·모노 | **JetBrains Mono** | 라벨(eyebrow)·AI 채팅·구조화 데이터·숫자 |

모노 폰트는 "기계(AI)가 읽는 페이지"라는 컨셉과 연결 — 장식이 아니라 의미.

**타입 스케일**: `--fs-xs .78 / sm .875 / base 1 / lg 1.125 / xl 1.375 / 2xl 1.75 / 3xl(clamp) / display(clamp 2.4–4.1rem)`
행간: 제목 1.15 · 본문 1.65 · 자간(제목) -0.03em.

---

## 4. 간격·모서리·그림자

- **간격**: 4/8pt 리듬 (`--sp-1`=4px … `--sp-16`=128px)
- **모서리**: `--r-sm 10 / md 16 / lg 24 / xl 32 / pill`(클라우드 로고와 통일감 위해 넉넉하게)
- **그림자**: `sm / md / lg` + 브랜드 그림자 `--shadow-brand`(버밀리언 글로우, CTA 버튼)

---

## 5. 시그니처 & 컴포넌트

- **시그니처**: 히어로의 "AI 답변 시뮬레이션" 카드 — AI 추천 답변에 `하오웹`이 버밀리언 하이라이트로 인용됨. 파는 걸 눈으로 시연.
- **하이라이트 시스템**: 히어로 취소선, `<mark>`, 강조 카드가 모두 버밀리언으로 "선택받음"을 표현.
- **버튼**: primary(버밀리언 필) / ghost(외곽선). 호버 시 -2px 상승.
- **카드**: 흰 표면 + 얇은 보더 + sm 그림자, 호버 시 상승·보더 강조.

---

## 6. 품질 기준 (지킬 것)

- 접근성: 포커스 링 유지, 본문 대비 ≥ 4.5:1, 시맨틱 마크업, `prefers-reduced-motion` 대응
- 반응형: 모바일 우선, 940 / 720 / 460px 브레이크포인트, 가로 스크롤 없음
- 모션: 150–300ms, transform·opacity만 애니메이션, 감속(ease-out)
- SEO/AEO/GEO: JSON-LD(Organization·FAQPage), 시맨틱 헤딩 위계, 메타태그 — 사이트가 스스로 시연

## 파일 구조
```
haoweb/
├─ index.html
├─ css/
│  ├─ tokens.css   ← 디자인 토큰(단일 소스). 여기만 고치면 전체 리스킨
│  └─ style.css    ← 토큰 참조 스타일
├─ js/main.js
├─ assets/logo.png ← 브랜드 로고
└─ DESIGN.md       ← 이 문서
```
> 색/폰트/간격을 바꾸려면 **`css/tokens.css`의 값만** 수정하세요. 전체가 따라 바뀝니다.
