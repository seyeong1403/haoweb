건별 유지보수

필요할 때만 요청하는 방식입니다.

  • 문구 수정
  • 이미지 교체
  • 페이지 추가
  • 배너·팝업 등록
  • 간단한 기능 수정
  • 오류 점검

월 정기 관리

  • 게시물 등록
  • 페이지 수정
  • 백업 점검
  • 기본 보안 업데이트
  • 링크와 폼 점검
  • 월간 작업 내역 정리

대상
