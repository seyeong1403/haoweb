방향부터 받아보세요.

업종과 목적을 알려주시면 하오웹이 홈페이지의 방향을 검토해 제안합니다. 신규 제작과 기존 홈페이지 리뉴얼 모두 신청할 수 있습니다.

신청하기

어떤 경우든

두 가지 상황 모두 신청 대상입니다.

신규 제작

새 홈페이지를 계획하고 있다면

제작 목적과 업종을 기준으로 어떤 구조와 콘텐츠가 필요한지, 어떤 플랜이 맞는지 검토해 제안합니다.

리뉴얼

기존 홈페이지가 있다면

현재 홈페이지 주소를 기준으로 무엇을 바꾸면 좋을지, 구조·콘텐츠·검색 관점의 개선 방향을 검토해 제안합니다.

신청 대상

이런 분께 필요합니다.

  • 홈페이지 제작을 검토 중이지만 무엇부터 해야 할지 모르는 분
  • 업체마다 다른 견적의 기준이 궁금한 분
  • 기존 홈페이지의 문제를 객관적으로 확인하고 싶은 분
  • 사이트 구조(사이트맵)를 먼저 보고 결정하고 싶은 분
  • 검색·AI 대응이 우리 홈페이지에 필요한지 궁금한 분
  • 내부 보고용 자료가 필요한 담당자

검토 항목

하오웹이 검토하는 것

  • 현재 홈페이지 또는 신규 제작의 목적
  • 권장 사이트맵
  • 핵심 콘텐츠 방향
  • 필요한 기능
  • 검색·AI(SEO·AEO·GEO) 검토 항목
  • 필요한 촬영·그래픽·콘텐츠 범위
  • 적합한 제작 플랜
  • 예상 제작 범위

고객이 제출하는 것

간단한 정보면 충분합니다.

  • 신규 제작인지, 리뉴얼인지
  • 업종과 회사·브랜드명
  • 홈페이지의 목적과 주요 고객
  • 현재 홈페이지 주소(리뉴얼의 경우)
  • 현재 느끼는 문제(선택)

하오웹이 확인하는 것

제출 정보를 바탕으로 검토합니다.

  • 업종 고객이 정보를 찾는 순서
  • 현재(또는 계획된) 구조와의 차이
  • 콘텐츠·기능의 우선순위
  • 검색·AI 관점의 점검 항목
  • 범위에 맞는 플랜

제안에 담기는 것

  • 홈페이지 방향 요약
  • 권장 사이트맵
  • 핵심 콘텐츠 방향
  • 검색·AI 검토 항목
  • 적합한 제작 플랜과 예상 범위
  • 상담 시 확인할 내용

제공하지 않는 것

  • 완성된 홈페이지 전체 디자인·개발 산출물
  • 검색 순위·AI 노출 등 성과 예측이나 보장
  • 타사 견적 비교·평가

제안 내용은 계약을 전제로 하지 않으며, 검토 후 진행 여부는 자유롭게 결정하시면 됩니다.

진행 과정

신청부터 제안까지

  1. 신청아래 폼을 작성해 신청합니다.
  2. 확인 연락담당자가 내용을 확인하고 필요한 경우 추가 질문을 드립니다.
  3. 검토업종·목적·현재 상태를 기준으로 방향을 검토합니다.
  4. 제안 전달검토 결과를 정리해 전달하고, 원하시면 상담으로 이어집니다.

자주 묻는 질문

신청 전 궁금한 것들

정말 무료인가요? 조건이 있나요?
신청과 제안 검토는 무료이며, 계약을 전제로 하지 않습니다. 제안을 받은 뒤 진행 여부는 자유롭게 결정하시면 됩니다.
자료가 없어도 신청할 수 있나요?
네. 업종과 목적만 알려주셔도 검토를 시작할 수 있습니다. 자료가 있으면 더 구체적인 제안이 가능합니다.
기존 홈페이지가 없어도 되나요?
네. 신규 제작의 경우 제작 목적과 참고 사이트를 기준으로 방향을 제안합니다.
제안을 받기까지 얼마나 걸리나요?
신청 내용과 검토 범위에 따라 다르며, 확인 연락 시 예상 일정을 함께 안내드립니다.
