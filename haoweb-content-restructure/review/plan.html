필요한 범위에 맞는
세 가지 플랜

플랜은 고객이 범위를 쉽게 가늠하기 위한 기준입니다. 실제 견적은 페이지 수와 콘텐츠·기능 범위에 따라 달라지며, 상담에서 확인한 범위로 안내합니다.

무료 시안·제안받기 제작비 결정 기준

플랜

어떤 자료가 있고,
무엇을 함께 만들어야 하는지에 따라.

베이직

정리된 자료가 있고,
필요한 구조를 정확하게 구축

회사소개서와 사진 등 기본 자료를 가지고 있고, 일반적인 기업·서비스 홈페이지가 필요하며, 복잡한 기능은 없는 회사에 적합합니다.

  • 주요 페이지 5~8개
  • 기본 정보구조 설계
  • 보유 원고 정리와 편집
  • 맞춤형 메인 디자인
  • 반응형 구현
  • 문의 폼 1개
  • 게시판 또는 공지 기능
  • 검색·AI 기본 구조
  • 기본 관리자 기능
  • 오픈 후 초기 오류 점검

프리미엄

콘텐츠부터 함께 만들고,
업종에 맞는 구성이 필요

콘텐츠가 충분히 정리되지 않았거나, 회사와 서비스 설명을 새로 만들어야 하거나, 업종에 맞는 차별화된 구성이 필요한 회사에 적합합니다.

  • 주요 페이지 10~15개
  • 고객·업종 조사
  • 사이트맵과 콘텐츠 기획
  • 주요 원고 작성
  • 맞춤형 전체 디자인
  • FAQ·칼럼·사례 구조
  • 문의 폼 또는 상담 흐름 설계
  • 검색·AI 기본 구조
  • 그래픽 제작
  • 촬영 기획 또는 선택 촬영
  • 관리자 기능
  • 오픈 후 안정화 지원

엔터프라이즈

브랜드·콘텐츠·촬영과
확장 기능을 함께 정비

서비스와 사업 영역이 많거나, 브랜드·콘텐츠·촬영을 함께 정리해야 하거나, 다국어·회원·외부 연동 같은 확장 기능이 필요한 회사에 적합합니다.

  • 20페이지 이상 또는 복합 구조
  • 심층 조사와 콘텐츠 전략
  • 전체 원고 기획·작성
  • 브랜드 시각체계 정리
  • 촬영과 그래픽·편집디자인
  • 다국어 구성
  • 회원·관리자·외부 연동
  • 고급 인터랙션
  • 검색·AI 확장 구조
  • 콘텐츠 운영 설계
  • 오픈 후 집중 안정화 지원
