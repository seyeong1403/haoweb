유지보수 문의

두 가지 방식

필요에 맞게 선택하세요.

건별 유지보수

필요할 때만 요청하는 방식입니다.

  • 문구 수정
  • 이미지 교체
  • 페이지 추가
  • 배너·팝업 등록
  • 간단한 기능 수정
  • 오류 점검

월 정기 관리

매달 정해진 범위 안에서 꾸준히 관리하는 방식입니다.

  • 정기 콘텐츠 업데이트
  • 게시물 등록
  • 페이지 수정
  • 백업 점검
  • 기본 보안 업데이트
  • 링크와 폼 점검
  • 월간 작업 내역 정리

대상

이런 경우 문의하세요.

  • 오픈 후 수정할 곳이 생겼을 때
  • 새 콘텐츠를 계속 올려야 할 때
  • 담당자가 바뀌어 관리가 어려울 때
  • 다른 곳에서 만든 홈페이지를 관리해야 할 때

하오웹이 만들지 않은 홈페이지는 현재 기술 환경과 관리자·서버 접근 권한을 확인한 뒤 유지보수 가능 여부와 범위를 안내합니다.
