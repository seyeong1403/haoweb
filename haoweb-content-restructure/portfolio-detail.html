프로젝트 상세

각 프로젝트의 상세는 아래 구조로 작성됩니다 — 무엇이 문제였고, 어떻게 설계했으며, 무엇이 만들어졌는지.

상세 구성

프로젝트 상세의 구조

  1. 개요고객·업종(공개 범위 내) / 제작 목적 / 제작 범위(실제 수행 범위만 표기)
  2. 배경과 문제어떤 상황에서 시작했고, 무엇이 문제였는지
  3. 정보구조어떤 판단 순서로 사이트맵을 설계했는지
  4. 콘텐츠 기획핵심 메시지와 페이지별 콘텐츠 방향
  5. 완성 화면Desktop·Mobile 실제 화면
  6. 브랜드·촬영·편집 연계함께 제작한 결과물(해당 시)

가상의 목업·성과 수치는 넣지 않으며, 실제 구축 화면과 확인된 내용만 게재합니다.
