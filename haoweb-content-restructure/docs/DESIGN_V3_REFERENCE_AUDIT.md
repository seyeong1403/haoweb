## 실측 (2026-07-23 라이브)

### 131 잇다소프트 (itddaa.com) — 주
- 디스플레이: 95~105px(뷰포트 961px 기준), weight 600, **자간 -4%**, 행간 0.8~1.0
- 서브 디스플레이 66px, 포트폴리오명 135~180px — **거대 타이포에만 원컬러**(일렉트릭 블루 #0600E8) 적용
- 한글 액센트 헤딩: `[감각]` `[마감]` — **브래킷 모티프**, 자간 +10%
- 라벨 12px/600, 내비 17px/400
- 섹션 여백 120px, 화이트·블랙 섹션 교차, 그림자 거의 없음(플랫)
- 버튼: 필(pill) 형태 "contact! start!"

### 112 도루코 (dorco.co.kr) — 보조
- Pretendard Std 전용, 헤딩 자간 -1.6~-3%
- 소형 라벨 12px/500 자간 -1.6%
- **레드는 화면 전체에서 라인·소형 포인트로만** — 배경·대면적 사용 없음
- 섹션 여백 69~139px, 화이트 기조 + 블랙 섹션 교차
