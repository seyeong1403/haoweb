# DECISIONS_NEEDED — 대표님 확인 필요 항목

> 아이디어시트·미팅에서 방향은 나왔지만 확정되지 않은 사항. 확정 전에는 공개 화면에 구체 수치·범위를 표기하지 않고 구조만 구현한다.
> 전사문에 음성 인식 오류가 있어 의미가 불분명한 부분도 임의 확정하지 않고 여기에 기록.

## 1. 무료 시안·제안서
- [ ] **무료 시안의 실제 제공 범위** — 실제 디자인 시안 전체 무료인지, 메인 방향/구조안 일부만인지. (미팅: "시안하고 제안서 받아보기, 무료 시안까지 무료로 줄게요" / 아이디어시트: "시안+제안서 받아보기 무료 · 아주 좋은 제안서: 시안/가격제안/aeo.geo.seo 개선/자동화칼럼")
- [ ] **무료 제안서의 결과물 구성** — 문서 형태·분량·전달 방식(PDF? 미팅?).
- [ ] "자동화칼럼"이 제안서에 포함되는 항목인지, 별도 서비스인지(전사 불명확).

## 2. 제작 플랜·가격
- [ ] **베이직/프리미엄/엔터프라이즈 각 가격과 포함 범위** — 미팅 중 금액 언급("10만원/2,000만원" 등)은 전사 오류 가능성이 높아 사용 불가. 실제 가격표 확정 필요.
- [ ] 플랜별 포함 항목(페이지 수·기능·콘텐츠·촬영 여부) 경계.

## 3. 글로벌 홈페이지
- [ ] **국가별(베트남·중국·태국·일본·영문) 독립 페이지 운영 여부** — 현재는 global.html 허브 내 섹션으로만 구현.
- [ ] **현지 사무소·현지 인력 보유 여부와 지원 범위** — 아이디어시트에 "현지사무소 보유를 강점으로 어필" 메모가 있으나 실제 보유 여부 미확인. 확인 전 공개 화면 표기 금지.
- [ ] 번역·감수 제공 방식(내부/외주/원어민 감수 여부).

## 4. 유지보수
- [ ] **유지보수의 GNB 위치** — 대표님 "유지보수 대메뉴로 나가야" 발언 vs 지시서 GNB 8개(고객지원 하위+Header 강조). 현재: 고객지원 하위 + 전체 메뉴 강조 노출로 구현. 대메뉴 승격 여부 확인.
- [ ] 유지보수 요금제·범위(월 정액? 건별?).

## 5. AI 콘텐츠·칼럼 운영
- [ ] **AI 콘텐츠의 실제 운영 방식** — 도구·검토 인력·발행 주기.
- [ ] **칼럼 발행 범위** — 하오웹 자체 칼럼 주기 / 고객사 대행 범위(미팅 "칼럼 100개씩" 발언은 목표치인지 실제 계획인지 불명확).
