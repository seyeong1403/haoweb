## 1. 포지셔닝
고객과 업종을 조사하고, 핵심 설명과 콘텐츠 구조, 신뢰 정보와 문의 흐름을 설계해 홈페이지를 제작하고 리뉴얼하는 **기업 홈페이지 제작 브랜드**.

- 핵심 사업: ①신규 홈페이지 제작 ②기존 홈페이지 리뉴얼
- 확장 서비스: 검색·AI 대응 / 글로벌 홈페이지 / 콘텐츠 제작 / 그래픽·편집디자인 / 촬영 / 콘텐츠 운영 / 유지보수
- 관계: 홈페이지 제작·리뉴얼 → 검색·AI 대응 → 글로벌 확장 → 원고·그래픽·촬영·운영 지원
- AI는 회사 정체성이 아니라 정보구조·콘텐츠 환경 변화에 대응하는 제작 요소로 설명한다.

## 4. 제작 플랜 (확정)
가격은 페이지·콘텐츠·기능 범위에 따라 달라지므로 **견적 상담**으로 안내한다. 페이지 수는 대표 기준이며 절대 제한이 아니다.

**베이직** — 기본 자료 보유, 일반적인 기업·서비스 홈페이지, 복잡한 기능 없음
5~8개 주요 페이지 / 기본 정보구조 / 보유 원고 정리·편집 / 맞춤형 메인 디자인 / 반응형 / 문의 폼 1개 / 게시판 또는 공지 / 검색·AI 기본 구조 / 기본 관리자 / 오픈 후 초기 오류 점검

**프리미엄** — 콘텐츠가 정리되지 않음, 설명을 새로 만들어야 함, 업종 맞춤 구성 필요
10~15개 주요 페이지 / 고객·업종 조사 / 사이트맵·콘텐츠 기획 / 주요 원고 작성 / 맞춤형 전체 디자인 / FAQ·칼럼·사례 구조 / 문의 폼·상담 흐름 / 검색·AI 기본 구조 / 그래픽 제작 / 촬영 기획 또는 선택 촬영 / 관리자 기능 / 오픈 후 안정화 지원

**엔터프라이즈** — 서비스·사업 영역이 많음, 브랜드·콘텐츠·촬영 통합, 다국어·회원·연동 필요
20페이지 이상 또는 복합 구조 / 심층 조사·콘텐츠 전략 / 전체 원고 기획·작성 / 브랜드 시각체계 정리 / 촬영·그래픽·편집디자인 / 다국어 / 회원·관리자·외부 연동 / 고급 인터랙션 / 검색·AI 확장 구조 / 콘텐츠 운영 설계 / 오픈 후 집중 안정화

## 17. 모션 시스템 (4유형)
A 텍스트 마스크 리빌 / B 이미지·화면 Clip Reveal / C 핵심 구간 Sticky / D Hover·Focus 마이크로.
이징 cubic-bezier(.16,1,.3,1). 숨김은 JS 클래스로만(무JS = 전부 표시), 안전망 2.6s, prefers-reduced-motion 비활성, bfcache 복귀 처리.
Mobile: 긴 Pin·Scrub 미사용, 정보는 누락하지 않음.

## 18. 반응형 규칙
1440/1280 데스크톱, 768 태블릿, 390/360 모바일 검수. 1280px 미만에서 모바일 메뉴, 480px 이하 헤더 CTA는 메뉴 안으로.
비교표는 Desktop 표 / Mobile 세로 블록·아코디언(가로 스크롤 금지).
플로팅 상담 버튼: 우하단, 최종 CTA·Footer·폼 입력 중·핵심 Sticky 장면에서 숨김(IntersectionObserver + 요소 겹침 판정).

## 19. 빌드 구조
- `src/site-config.js` — 회사·연락처·개인정보·폼 endpoint 단일 소스
- `src/data/portfolio.json`, `src/data/interview.json` — 실제 데이터(현재 비어 있음)
- `_build/compose.ps1` — 공통 크롬 + 프래그먼트 → 페이지
- `_build/make_dist.ps1` — `dist/`(공개용, docs·archive·템플릿·데이터 없는 상세 제외) / `review/`(대표님 검토용 전체)
