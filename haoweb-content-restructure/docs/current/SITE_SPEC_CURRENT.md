## 1. 포지셔닝
검색엔진과 AI가 함께 웹을 읽는 시대에, 고객과 업종을 조사하고 **메시지·콘텐츠 구조·AI 가시성(SEO·AEO·GEO)**을 함께 설계해 홈페이지를 제작·리뉴얼하는 브랜드.

- 핵심 사업: ①신규 홈페이지 제작 ②기존 홈페이지 리뉴얼
- 핵심 차별점: **AI 가시성** — 검색엔진과 AI가 발견·인용할 수 있는 구조를 제작 단계부터 설계
- 확장·연계: 글로벌 홈페이지(현지 사무소) / AI 콘텐츠 / 그래픽·편집디자인 / 스튜디오·촬영 / 유지보수
- AI는 회사 정체성이 아니라 정보구조·콘텐츠 환경 변화에 대응하는 제작 요소로 설명한다.

- 모바일: 동일 7그룹 구조. Header/모바일 CTA 모두 무료 시안·제안받기 Primary.

## 5. 무료 시안·제안받기 (Header 최강 CTA)
- 대상: 신규 제작 검토 기업 / 리뉴얼 필요 기업 / 방향·범위를 먼저 확인하려는 담당자
- 신청 정보: 회사명·담당자·연락처·이메일·신규/리뉴얼·현재 URL·주요 사업·제작 목적·필요 기능·보유 자료·희망 일정·예산 범위
- 제공(12항목): 신규 제작 목적 또는 현재 홈페이지 검토 / 권장 사이트맵 / 메인페이지 핵심 콘텐츠 방향 / 주요 CTA·문의 흐름 / 필요 기능·콘텐츠 / SEO·AEO·GEO 개선 항목 / 적합한 제작 플랜 / 예상 제작 범위 / **견적 방향** / **메인 첫 화면 시안 1종** / **자동화 또는 정기 운영 가능한 칼럼 주제안** / 콘텐츠 발행·운영 방향
- **견적 방향** = 확정 금액 아님. "프로젝트의 페이지·콘텐츠·기능 범위를 기준으로 예상 제작 범위와 견적 방향을 함께 안내." **자동화 칼럼** = 자동 생성 보장 아님. "기업 서비스·고객 질문 기반으로 정기 운영 가능한 칼럼 주제·콘텐츠 운영 방향 제안."
- 결과물: PDF 또는 온라인 제안 화면, 영업일 3~5일 이내, 필요 시 추가 질문
- 미포함: 전체 페이지 디자인 / 전체 원고 / 개발 소스 / 원본 디자인 파일 / 반복 수정 / 실제 배포
- `?type=renewal`로 리뉴얼 신청 진입 가능. 고객지원 2Depth "무료 제공 서비스"로도 연결.

## 6. AI 가시성 (핵심 차별점)
- **search-ai(소개)**: AI 검색 대응 개요, 왜 필요한지, 하오웹의 기본 포함/확장 범위 구분
- **SEO**: 검색엔진이 발견·색인하는 구조(정보구조·시맨틱·속도·URL·제목/설명)
- **AEO**: 고객 질문에 답하는 콘텐츠 → 검색엔진·일부 생성형 AI가 파악·참고하기 쉬운 질문/답변 구조(인용 보장 표현 금지)
- **GEO**: 회사·서비스·근거의 관계 정리 → AI가 맥락·신뢰 파악
- **AI 콘텐츠(ai-content)**: 핵심 문장·원고·전문 콘텐츠 8공정. content-production/operation 통합 흡수
- 세 페이지(SEO/AEO/GEO)는 **문제 정의·범위·산출물·CTA가 서로 구분**되어야 하며 동일 문구 복제 금지.
- 표현 기준: 노출·순위·AI 답변 **보장 금지**. "검색엔진과 일부 생성형 AI가 공개 웹페이지 정보를 수집·참고할 수 있도록 관계를 명확히 구성합니다."

## 7. 글로벌 홈페이지 + 국가 페이지
- **확정 사실: 하오웹은 베트남·중국·태국에 현지 사무실을 운영한다.** → 핵심 신뢰 요소로 배치.
- **권장 표현(고정)**: "하오웹은 베트남·중국·태국 현지 사무실을 운영하고 있습니다. 국내 기획·디자인·개발 과정과 현지 커뮤니케이션을 연결해 국가별 홈페이지 제작과 운영을 지원합니다." / 번역·감수: "번역과 언어 감수는 프로젝트의 국가와 콘텐츠 범위에 따라 진행 방식을 구성합니다."
- **확정 사실 이상으로 만들지 않는다(미확인)**: 현지 직원이 번역/원어민 감수/국가별 검색 환경 분석/현지 법률 검토를 수행한다는 서술, 현지 사무실이 전문 SEO를 수행한다는 서술, 일본·영어권에 협력 전문가가 있다는 서술, 국가별 전문 번역 인력 상주 — 모두 삭제·미사용.
- 하오웹 제공: 다국어 정보구조 / 언어별 페이지 / 반응형·퍼블리싱 / 언어별 메뉴·URL / 해외 문의 폼·언어 전환 / 기본 hreflang / 언어별 제목·설명 / 다국어 입력 구조 / 번역·감수 진행 방식 구성 / 국가별 표현 검토
- 협의 후 진행: 국가별 도메인·서브디렉터리 / 서버·호스팅 / 다국어 운영
- 국가 페이지:
  - **베트남·중국·태국** — `gl-badge` 현지 사무실(red). 현지 사무실 운영 + 국내 제작과 현지 커뮤니케이션 연결까지만 표현.
  - **일본·영어권(미국·글로벌)** — 배지 없음. **현지 사무실이 있다고 표현하지 않으며, 협력 전문가 존재도 단정하지 않는다.** 시장 특성(격식·바이어 기준) + 번역·감수 범위 구성으로 서술.
- 정확한 주소·전화·담당자명·직원 인원·번역 담당자명은 만들지 않는다(→ PENDING_FACTS).

## 8. 제작 플랜 (확정)
가격은 페이지·콘텐츠·기능 범위에 따라 달라지므로 **견적 상담**으로 안내. 페이지 수는 대표 기준.

**베이직** — 기본 자료 보유, 일반 기업·서비스, 복잡 기능 없음. 5~8p / 기본 정보구조 / 보유 원고 정리 / 맞춤 메인 디자인 / 반응형 / 문의 폼 / 게시판·공지 / AI 가시성 기본 구조 / 기본 관리자 / 초기 오류 점검
**프리미엄** — 콘텐츠 미정리, 설명 신규 작성, 업종 맞춤. 10~15p / 고객·업종 조사 / 사이트맵·콘텐츠 기획 / 주요 원고 / 맞춤 전체 디자인 / FAQ·칼럼·사례 구조 / 상담 흐름 / AI 가시성 기본 / 그래픽 / 촬영 기획·선택 촬영 / 관리자 / 안정화
**엔터프라이즈** — 서비스 다수, 브랜드·콘텐츠·촬영 통합, 다국어·회원·연동. 20p+ / 심층 조사·콘텐츠 전략 / 전체 원고 / 브랜드 시각체계 / 촬영·그래픽·편집 / 다국어 / 회원·연동 / 고급 인터랙션 / AI 가시성 확장 / 운영 설계 / 집중 안정화
