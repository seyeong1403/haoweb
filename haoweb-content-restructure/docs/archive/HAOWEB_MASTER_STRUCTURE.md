## 1. 브랜드 정의

**핵심 정의**
하오웹은 신규 홈페이지 제작과 기존 홈페이지 리뉴얼을 중심으로, 고객이 회사를 이해하고 신뢰한 뒤 문의할 수 있도록 메시지, 콘텐츠, 정보구조와 기능을 설계하는 홈페이지 제작 브랜드다.

**보조 정의**
검색엔진과 생성형 AI가 회사·서비스·콘텐츠의 관계를 이해하기 쉬운 구조를 반영하고, 브랜딩·촬영·그래픽디자인·편집디자인·콘텐츠 운영까지 연결한다.

**핵심 목적** — 문의가 들어오는 홈페이지.
단, "문의를 자동으로 만든다·성과 보장" 표현 금지. 허용 표현:
- 문의로 이어질 수 있는 구조
- 고객의 판단과 문의를 고려한 홈페이지
- 필요한 정보를 확인하고 상담할 수 있는 동선
- 고객이 이해하고 신뢰하도록 설계한 콘텐츠

**제작 철학(미팅 방향 1)** — 원고를 정리해 예쁘게 배치하는 일은 제작이 아니다. 하오웹이 먼저 찾아내는 것:
① 고객 기업이 전달해야 할 핵심 메시지 ② 방문자가 확인해야 할 정보 ③ 방문자의 판단 순서 ④ 신뢰를 만드는 근거 ⑤ 문의를 결정하게 하는 요소 ⑥ 고객이 실제로 도달해야 할 목적.

**모든 페이지가 답해야 할 5문(미팅 방향 2)**
1. 이 페이지는 누구를 위한 것인가?
2. 고객은 어떤 문제를 가지고 들어오는가?
3. 어떤 내용을 확인하면 신뢰하는가?
4. 왜 하오웹에 문의해야 하는가?
5. 다음 행동은 무엇인가?

## 2. 사업 방향

| 축 | 위치 | 설명 |
|---|---|---|
| **신규 홈페이지 제작** | 인바운드의 중심 · 메인 Primary 경로 | 업종별(기업·병원·변호사·쇼핑몰·프랜차이즈) + 랜딩·앱. 하나의 "홈페이지 제작센터" 허브로 묶는다. 업종별 별도 메인 홈페이지는 만들지 않는다(대표님 지시: "하나의 메인"). |
| **홈페이지 리뉴얼** | 별도 핵심 사업 · 광고용 랜딩 겸용 | 디자인 교체가 아니라 메시지·정보구조·콘텐츠 순서·모바일·문의 동선·SEO/AEO/GEO·FAQ·칼럼·사례·내부 링크·운영 구조까지. Meta 광고·아웃바운드 영업의 착지 페이지로 사용 가능해야 함. |
| **AI 검색 대응(SEO·AEO·GEO)** | 독립 서비스이자 모든 제작의 기본 반영 | 신규·리뉴얼 양쪽에 통합 적용. 별도 광고 상품처럼 팔지 않되, 서비스 설명 페이지는 갖는다. 결과 보장 표현 전면 금지. |
| **글로벌 홈페이지** | 대메뉴급 서비스 · 허브만 우선 | 베트남·중국·태국·일본·영문. 국가별 독립 페이지 여부 미확정 → global.html 허브 안에 국가 섹션 구조만 준비(DECISIONS_NEEDED). |
| **콘텐츠 제작** | 제작을 지원하는 통합 서비스 | 메시지·원고, AI 활용 전문 콘텐츠(사람 검토 공정 명시), 칼럼·FAQ·사례, 그래픽·편집디자인, 카탈로그·브로슈어·상세페이지, 기업·제품·공간·스튜디오 촬영. 별도 회사처럼 분리하지 않고 항상 홈페이지와 연결해 설명. |
| **유지보수(운영)** | 오픈 이후 사업 | 수정·업데이트 + 칼럼·FAQ·사례 등 콘텐츠 운영. GNB 위치는 미확정(헤더 보조 or 고객지원 하위) → DECISIONS_NEEDED. |
| **무료 시안·제안서** | 문의 유도 장치(리드젠) | 신규·리뉴얼 공용. 검토 항목 구조는 확정, 실제 무료 제공 범위는 미확정 → 구조만 구현, 범위는 TODO/DECISIONS_NEEDED. |

## 3. 고객 유형 (전 콘텐츠·CTA의 기준)

**A. 신규 제작 고객** — 홈페이지가 없는 기업 / 새 사업·브랜드 / 별도 서비스 사이트 필요 / 업종별(기업·병원·변호사·쇼핑몰·프랜차이즈) / 랜딩·앱 필요.
→ 경로: index → website → 업종 상세 → plan/free-proposal → inquiry

**B. 리뉴얼 고객** — 디자인 노후 / 모바일 불편 / 설명 부족 / 문의 동선 불명확 / FAQ·칼럼·사례 부재 / 검색·AI가 이해하기 어려운 구조 / 부분·전체 판단 어려움. 업종 제한 없음(기업만 아님).
→ 경로: index → renewal → diagnosis 또는 renewal-proposal → inquiry

**C. 글로벌 고객** — 해외 진출 준비 / 다국어 필요 / 해외 고객 대상 설명 / 국가별 검토 필요.
→ 경로: index(허브 소개) → global → free-proposal/inquiry

**D. 콘텐츠 제작 고객** — 원고 부족 / 촬영 필요 / 그래픽·편집 필요 / 카탈로그·상세페이지 동시 제작 / 콘텐츠 지속 운영.
→ 경로: index(§08) → content-production → 세부(ai-content/graphic-design/studio) → inquiry

**E. 유지보수 고객** — 오픈 후 수정·업데이트 / 콘텐츠 추가 / 지속 관리.
→ 경로: header·footer·final CTA → maintenance → inquiry(유지보수 선택)

## 5. 금지·주의 표현 (전 페이지 공통)

- 보장 금지: AI 추천/검색 순위/AI 인용/노출 증가율/문의 증가율 보장.
- "95% 이상 리뉴얼 대상"(미팅 발언) — 검증되지 않은 수치, 공개 화면 사용 금지. 대체: "많은 기존 홈페이지가 디자인 외의 구조적 문제를 가지고 있습니다 / 겉보기에는 정상이어도 콘텐츠와 검색 구조는 오래되어 있을 수 있습니다 / 현재 환경에 맞는지 별도의 점검이 필요합니다."
- 공포 마케팅·과장 긴급성 금지(긴급성은 사실 기반으로만).
- 개발중 문구 금지: 준비 중/Coming soon/추후 입력 예정/등록된 사례 없음 등.
- 임의 생성 금지: 고객사명·실적·성과수치·후기·팀원·가격·기간·무료 범위·해외 사무소/인력.
- AI 콘텐츠를 "자동 생성·대량 생산"으로 표현 금지 → 분석→주제→자료→초안→사실확인→담당자 검토→편집·발행→업데이트 공정으로 설명.
- 의료(병원) 콘텐츠: 치료 후기·효능 단정 금지(의료광고 심의 고려) — 기존 원칙 유지.
