# CONTENT_PLAN_INTERNAL — 자체 콘텐츠 발행 계획 (내부 관리 · 공개 금지)

> 제목만 있는 예정 주제는 **공개 화면에 게재하지 않는다**(가짜 목록·빈 상세 링크 금지).
> 실제 본문이 작성된 글만 columns.html에 추가한다.

## 현재 공개 중
| 상태 | 제목 | 파일 |
|---|---|---|
| 발행 | 기업 홈페이지 제작 전에 준비할 자료 | column-detail.html |

## 작성 예정 주제(비공개)
### 제작 준비
- 자료가 부족해도 시작하는 순서

### 업종별 구조
- 병원 홈페이지에서 환자가 먼저 확인하는 정보
- 변호사 전문 분야 페이지 구성
- 프랜차이즈 가맹 문의로 이어지는 페이지 구성

### 리뉴얼
- 리뉴얼 시 유지할 콘텐츠와 다시 쓸 콘텐츠
- 디자인만 바꾸면 남는 문제

### 검색과 AI
- SEO·AEO·GEO는 어떻게 다른가
- 검색 시스템이 해석하기 쉬운 FAQ 작성법
- 검색 노출을 보장할 수 없는 이유

### 운영
- 오픈 후 90일 동안 운영할 콘텐츠
- 홈페이지 비용을 결정하는 요소

## 발행 원칙
- 하오웹이 검색·AI 대응과 콘텐츠 운영을 서비스로 제공하려면 **자체 사이트에도 실제 칼럼·FAQ·사례가 있어야 한다.**
- 각 글은 실제 업무 경험을 근거로 작성한다(일반론 나열 금지).
- 발행 시 columns.html 목록에 추가하고, 관련 서비스 페이지와 내부 링크로 연결한다.

## 공지사항
- 실제 공지가 1건 이상 확보되기 전까지 notice.html은 GNB 비노출(현재 상태).
