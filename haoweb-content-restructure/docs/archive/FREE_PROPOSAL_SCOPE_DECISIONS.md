# FREE_PROPOSAL_SCOPE_DECISIONS — 무료 제안 서비스 범위 확인 (2026-07-23 · 대표님 확인 필요)

> 무료 제안은 하오웹의 핵심 문의 유도 서비스지만 실제 제공 범위가 미확정이다.
> **확정 전까지 공개 화면에서는 아래 "현재 허용 문구"만 사용한다.**

## 확인 필요 항목

### 서비스 정의
- [ ] 공개 서비스 명칭(현재 "무료 제안받기" / 아이디어시트에는 "무료 시안·제안서")
- [ ] **무료 시안(디자인) 제공 여부** — 제공하지 않으면 "시안" 표현 자체를 쓰지 않는다
- [ ] 디자인 시안 제공 시: 제공 범위(메인 1개 / 서브 포함), 형식, 수정 가능 여부·횟수

### 결과물
- [ ] 권장 사이트맵 제공 수준(전체 / 대메뉴 수준 / 구두 안내)
- [ ] 콘텐츠 방향 제안 수준(항목 나열 / 페이지별 방향 / 원고 샘플)
- [ ] SEO·AEO·GEO 검토 범위(체크 항목 / 진단 리포트)
- [ ] 제작 플랜·가격 제안 포함 여부
- [ ] 결과물 전달 형식(PDF / 링크 / 미팅 구두)
- [ ] 신규 제작과 리뉴얼의 결과물 차이

### 운영 정책
- [ ] 신청 대상(제한 없음 / 업종·규모 조건)
- [ ] 신청자 선별 여부와 기준
- [ ] 제공 일정(신청 후 며칠)
- [ ] 제공 횟수(1회 / 재신청 가능)
- [ ] 계약과의 관계(계약 전제 여부 — 현재 "계약을 전제로 하지 않습니다" 문구는 전면 제거함)
- [ ] Meta 광고 유입 고객의 처리 방식(별도 랜딩/별도 응대 여부)

## 현재 허용 문구(확정 전)
> "신청 내용을 바탕으로 현재 상황과 필요한 제작 범위를 검토합니다. 실제 검토 항목과 전달 방식은 신청 내용을 확인한 뒤 안내합니다."

## 확정 전 사용 금지 표현
- 무료 디자인 시안을 제공합니다
- 권장 사이트맵을 제공합니다 (→ "검토할 수 있는 항목"으로만)
- 내부 보고용 문서를 제공합니다
- 확정된 결과물 목록 / 확정된 전달 일정 / 수정 가능 횟수
- 모든 신청자에게 동일한 범위를 제공한다는 표현
- 계약을 전제로 하지 않습니다(운영 정책 미확정)
