# FIVEPRO_REFERENCE_AUDIT

> 마스터 디자인 레퍼런스: https://fivepro.co.kr (대표님 지정 + 세영님 확정)
> 라이브 직접 실측(DOM/computed style, 2026-07-23). **구성 원리·타이포·컴포넌트 문법만 차용** — 문구·이미지·로고·코드·고객사명 복사 금지.

## 1. 전체 인상
- **화이트 기반의 밝고 신뢰감 있는 B2B 톤.** 다크 아님. 배경 #FFF, 본문 #333, 헤딩 near-black(#0C0C0C·#1B1B1B).
- 폰트 **Pretendard 단일**(하오웹과 동일 — 폰트 변경 불필요).
- **실제 프로젝트 결과물(스크린샷·영상)이 화면의 주인공.** 히어로부터 이미지·영상 다량(video 9, img 138). 추상 그래픽 없음.
- 페이지 길이 ≈ 20.5 스크린. 섹션이 많지만 리듬이 단순해 부담 없음.

## 2. 컬러 (실측)
- 배경: #FFFFFF / 본문: #333 / 헤딩: #0C0C0C
- **포인트 블루: #235BF5**(주 버튼·강조 문구), 보조 진블루 #274BB1
- **세컨더리 버튼: 연블루 틴트 배경(#DDE9F7) + 블루 텍스트** ← 시그니처. 흰 배경 위 소프트 대비.
- 다크 밴드(성과 수치 구간): 어두운 배경 + 흰 대형 숫자(47px).
- **하오웹 적용 결정**: 포인트는 파이브프로 블루가 아니라 **HAOWEB Red #E83817 유지**(기존 원칙). 문법만 이식: 주 버튼=레드 솔리드, 세컨더리=레드 틴트(#FDEAE4 계열)+레드 텍스트. ※ 대표님이 블루 전환을 원하실 가능성 → DECISIONS_NEEDED에 추가.

## 3. 타이포그래피 (실측)
- 헤딩 굵기 **700 고정**(얇은 헤딩 없음 — 상공과 정반대). 섹션 제목 30~47px, FAQ·대형 제목 47px.
- 강조 리드 문구는 **블루(포인트 컬러) 26px 700** — "내 브랜드에 꼭 맞는 웹사이트, 고민이신가요?" 식 질문형.
- 포트폴리오 카드 제목 24px 700.
- 본문 16px, 부가정보는 회색(#B1B1B1 급) 처리.
- **하오웹 적용**: 헤딩 700, 섹션 제목 스케일 상향(현 중립 CSS보다 큼), 질문형 리드에 레드 포인트.

## 4. 시그니처 컴포넌트
1. **인용형 포트폴리오 카드**: `"고객사 | 프로젝트를 한 문장으로 말하는 스토리"` — 스크린샷 위에 따옴표 인용문 24px. 사례가 '작업물'이 아니라 '이야기'로 읽힘. → 하오웹 포폴 등록 구조에 그대로 이식 가능(실제 고객명 확보 전에는 업종 표기).
2. **연도 타임라인**: 연혁·실적을 연도(32px 700) 축으로 나열.
3. **성과 수치 다크 밴드**: 흰 47px 숫자 3개(100% · 2.7배 · 98%). ⚠️ 하오웹은 **검증 수치가 없으므로 숫자 밴드는 도입 보류** — 대신 다크 밴드 스타일만 Final CTA에 사용(이미 cta-band 있음).
4. **질문형 CTA 섹션**: 포인트 컬러 질문 한 줄 + 버튼 2개(솔리드+틴트).
5. **라운드 카드**: border-radius **20px가 지배적**(65회), 버튼 7~10px. 부드럽고 현대적인 인상.
6. **프로세스 3단**: "빠르고 효율적인 구축 / 디자인·개발 연결 / 확장 가능" — 각 30px 제목 + 설명.

## 5. 헤더·GNB
- 흰 배경 85px, relative(스크롤 시 재노출 방식 별도). 네비: 제작 서비스/포트폴리오/고객 인터뷰/제작 플랜/블로그/유지보수/고객지원 + **견적문의 블루 버튼**.
- 하오웹 GNB 8과 거의 동일 구조 → 그대로 호환. 견적문의=제작 문의(레드 버튼).

## 6. 모션 (관찰)
- 스크롤 리빌은 절제된 페이드/슬라이드. 포폴 캐러셀·영상 재생이 주된 움직임. 글리치·패럴랙스 없음.
- 하오웹 적용: 현 중립 구조 유지 + 필요 최소(호버 lift, 부드러운 transition)만. 별도 JS 모션 도입 안 함(§17 원칙 유지).

## 7. 하오웹 이식 계획 (CSS 층만 교체)
| 파이브프로 문법 | 하오웹 적용 |
|---|---|
| 화이트 + near-black + 블루 포인트 | 화이트 + near-black + **레드 포인트**(원칙 유지) |
| 틴트 세컨더리 버튼 | 레드 틴트(#FDEAE4)+레드 텍스트 |
| 라운드 20px 카드 | path·col-block·폼·표에 12~20px 라운드 |
| 헤딩 700 · 큰 스케일 | h1 3.4rem·h2 2.2rem급 700 |
| 질문형 포인트 리드 | eyebrow → 레드 질문형 리드 스타일 추가 |
| 인용형 포폴 카드 | portfolio 등록 구조에 인용 스타일 반영 |
| 다크 성과 밴드 | cta-band에 스타일만(수치는 미도입) |
| 실제 결과물 이미지 중심 | 실 스크린샷 확보 후 적용(TODO) — 데모 UI 재사용 가능 |

## 8. 복사 금지 (준수 확인)
- 파이브프로 고유 문구·고객사명(dkb하우스·빗썸 등)·이미지·영상·로고·코드 — 사용 안 함.
- 성과 수치(100%·2.7배·98%) — 하오웹은 검증 수치 없음 → 미도입.
- 블루 컬러 — 하오웹 레드 유지(전환 여부는 대표님 확인 항목).
