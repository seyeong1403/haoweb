## website.html — 홈페이지 제작센터 (허브·신규)
- 목적: A고객이 자신에게 맞는 제작 유형을 찾고 세부로 이동.
- 대상: A. / 핵심 메시지: "업종과 고객이 다르면, 홈페이지의 구조도 달라야 합니다."
- 섹션(13): 제작의 목적(문의 관점) → 업종·고객에 따라 구조가 달라지는 이유 → 제작 유형 안내 → 기업·병원·변호사·쇼핑몰·프랜차이즈(각 요약+링크) → 랜딩·앱 → 제작에 필요한 콘텐츠 → 자료 부족 지원(→content-production) → SEO·AEO·GEO 기본 반영 → 제작 플랜 → 제작 과정 → FAQ → 무료 시안·제안서 → 제작 문의.
- 공통 설명(자료 전환·브랜드 연계·프로세스·비용)은 이 허브+전용 페이지가 담당, 업종 페이지 중복 제거.
- CTA: 업종 상세 / free-proposal / inquiry.

## company / hospital / lawyer / shop / franchise / landing / app — 업종 7종 (수정)
- 목적: 해당 업종 고객의 문제와 하오웹의 설계 방식 설득.
- 공통 골격(§14 12문 답): 도입(누구·문제) → 이런 곳에 → 업종 고객 탐색 과정 → 업종 핵심 문제 → 하오웹 접근(업종 설계 방식) → 화면 구조(기존 데모 유지) → 필요한 콘텐츠·기능 → 준비 자료(간단)+자료 부족 시(→content-production) → 비용·기간 요인 2~3줄(→price-guide) → 업종 특화 FAQ → 관련 칼럼 → 업종 특화 CTA(+free-proposal).
- 제거: 공통 프로세스·자료전환·브랜드통합·검색AI 상세(허브로 이동), 포폴/인터뷰 슬롯(링크 대체).
- 병원 주의: 치료 후기·효능 단정 금지.
