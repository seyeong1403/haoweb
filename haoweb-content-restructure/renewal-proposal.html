개선안 신청하기

개선안 구성

개선안에 담기는 것

  • 현재 홈페이지의 문제 요약
  • 권장 사이트맵(메뉴·페이지 구조)
  • 핵심 콘텐츠 방향
  • 검색·AI(SEO·AEO·GEO) 검토 항목
  • 필요한 제작 범위(콘텐츠·촬영·기능)
  • 적합한 제작 플랜

신청 대상

이런 경우 신청하세요.

  • 리뉴얼을 검토 중인데 어디서부터 시작할지 모르는 경우
  • 디자인만 바꿀지 전체 구조를 바꿀지 판단이 필요한 경우
  • 내부 보고를 위해 개선 방향 자료가 필요한 경우
  • 여러 업체를 비교하기 전에 기준을 갖고 싶은 경우

진행 과정

신청부터 전달까지

  1. 신청현재 홈페이지 주소와 기본 정보를 알려주세요.
  2. 검토6가지 진단 영역을 기준으로 현재 상태를 살펴봅니다.
  3. 개선안 작성문제 요약과 개선 방향, 권장 구조를 정리합니다.
  4. 전달·상담개선안을 전달하고, 원하시면 상담으로 이어집니다. 계약을 전제로 하지 않습니다.
