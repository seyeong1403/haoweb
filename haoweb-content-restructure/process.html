아홉 단계

화면을 먼저 그리지 않습니다. 이해하고, 설계하고, 만들고, 오픈 후에도 함께 운영하는 순서로 진행합니다.

  1. 상담업종·목표·현재 상태를 확인합니다. — 고객 확인: 목표와 범위 / 산출물: 상담 정리
  2. 목표와 고객 분석홈페이지가 도달할 목적과, 방문할 고객이 확인하려는 정보를 조사합니다. — 산출물: 핵심 메시지·고객 질문 정리
  3. 정보구조메뉴와 페이지 구조를 방문자의 판단 순서로 설계합니다. — 고객 확인: 사이트맵 / 산출물: 사이트맵·페이지 목록
  4. 콘텐츠 기획페이지별로 무엇을 말할지 정하고 원고를 준비합니다. — 고객 확인: 페이지별 콘텐츠 방향 / 산출물: 원고
  5. 브랜드·촬영·그래픽필요한 경우 로고·촬영·편집물을 함께 제작합니다. — 산출물: 이미지·그래픽 자산
  6. 디자인브랜드 기준에 맞춰 화면을 설계합니다. — 고객 확인: 시안 / 산출물: 디자인
  7. 개발반응형으로 구현하고 필요한 기능을 만듭니다. — 산출물: 작동하는 홈페이지
  8. 검수와 오픈내용·동작·모바일을 점검하고 공개합니다. — 고객 확인: 최종 검수 / 산출물: 오픈
  9. 콘텐츠 운영칼럼·FAQ·사례를 쌓고 유지보수로 관리합니다. — 산출물: 운영 콘텐츠
